Each node represents a Telegram chat. When a message from one chat gets forwarded to another an edge is formed which decays over time. Repeated forwards strengthen the edge. Nodes are clustered according to Edge widths reprent their strength. Node size represent the node's degree. Colors indicate the dominant language in the chat. The visualization uses sigma.js to render the graph using a force layout, where connected nodes attract and non-connected nodes repell each other.
